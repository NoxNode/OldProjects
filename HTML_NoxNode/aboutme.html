About Me

Hello, I'm Noxide. I've been programming games since the Summer of 2012 when I was 14. I like games and pretty much anything that has to do with computers. I like doing things as low level as efficiently possible, meaning I want as much control over the hardware as possible, but I also want to be able to work fast and have my code be cross platform. Because of this I'll be using my own custom engines for the games I make.
